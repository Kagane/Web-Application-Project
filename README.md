## Web-Application-Project

Contains 3 roles: student, instructor and training provider

# Use Case
All:
    -Login
    -Sign up
    -Logout
    
Training Provider:
    -Add courses
    -Update courses
    -Delete courses
    
Student:
    -Register courses
    -View enrolled courses
    -Start/Continue Ongoing courses
    -Submit feedback
    -View certificate of completion
    -View profile
    
Instructor:
    -Update availability
